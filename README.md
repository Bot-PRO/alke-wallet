# wallet-alke

## Primera pagina web

creando una fintech basica con funcionalidad con JS/jQuery, y estilos con CSS/Bootstrap.

## Estado del Proyecto

Este proyecto es un **protitipo**: el cual implementa las funciones basica de una billetera digital(transacciones, transaferencias,historial, etc.), del lado del frontend, con localStorage.  

La paleta de colores actual es basica. A futuro busco rediseñarla hasta que se acerque mas a una fintech real.
